Task: Vehicle Route Optimization

Objective:

Develop a Python script to optimize vehicle routes for 3 drivers and 17 deliveries using OR-tools. The script should process incoming data, organize deliveries into optimized routes based on pickup and drop-off locations, time windows, and prepare a response in JSON format with sequence.

Note:

Drivers are in the same postcode area for collection.

Input:

Sample data in a structured format containing information about each delivery, including order ID, collection postcode, collection time window, and stop postcode.

Order ID Collection Postcode Collection From (HH:MM) Collection To (HH:MM) Stop Postcode
1 EC4A 1AA 08:30:00 09:00:00 SW1A 1AA
2 EC4A 1AA 08:30:00 09:00:00 EC1A 1AA

Output:

A JSON response that organizes the deliveries into optimized routes for each driver.

Task Details:

  1. Data Processing:
    • Parse the incoming JSON data to extract pickup and drop-off requests.
    • Organize the requests into a suitable data structure for route optimization.
  2. Route Optimization:
    • Utilize Model to define and solve the vehicle routing problem.
    • Implement constraints such as vehicle capacities, time windows, and pickup/drop-off locations.
    • Optimize routes for each driver based on their depot location and time windows.
  3. Response Preparation:
    • Prepare a JSON response that includes optimized routes for each driver.
    • Format the response with sequence according to the specified structure, including order locations and request IDs.
  4. Testing and Validation:
    • Test the script with sample data to ensure it generates valid and optimized routes.
    • Validate the response format against provided examples.
  5. Documentation and Reporting:
    • Document the code thoroughly, including explanations of key components and functions.
    • Provide clear instructions on running the script and interpreting the results.
    • Include a report summarizing the approach, challenges, and solution performance.
