Task: Vehicle Route Optimization

Objective:

Develop a Python script to optimize vehicle routes for 17 deliveries (from collections to stops) using OR-tools. The script should process by get_json_data, and organize deliveries into optimized routes based on pickup and drop-off locations, time windows, and prepare a response in JSON format with sequence in result optimize_stops.

Facts:

  • Unlimited drivers
  • Allow Waiting time 30 mins.
  • Max hours for each journey and route 8hr

Input:

jsonData in a structured format containing information about each delivery, including order ID, collection postcode, collection time window, stop postcode, latitude, and longitude.

Output:

A JSON response in result of optimize_stops that organizes the deliveries into optimized routes.
